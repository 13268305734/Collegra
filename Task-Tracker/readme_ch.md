# 任务追踪器
[English version](README.md)

任务追踪器是Collegra的轻量级模块，帮助大学生高效管理日常学习任务和截止日期。

## 已实现功能
- **任务管理**：创建、完成和删除带标题、描述和截止日期的任务
- **优先级系统**：将任务标记为高/中/低优先级
- **可视化组织**：颜色区分的优先级标识和独立的活动/已完成任务视图
- **响应式设计**：在桌面和移动设备上都能良好工作

## 贡献指南
扩展本模块需遵循：
1. 在`app.py`中添加新功能
2. 保持所有HTML模板继承自`base.html`
3. 维护中英文双版本模板
4. 提交pull request
